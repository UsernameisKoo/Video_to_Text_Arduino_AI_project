# Video_to_Text_Arduino_AI_project

### Python packages that must be installed
+ EasyOCR
  
    pip install easyocr
  
+ OpenCV
  
    pip install opencv-python
  
+ Matplotlib
  
    pip install matplotlib

+ PySerial

  pip install serial

### Arduino library that must be installed
+ LiquidCrystal_I2C
